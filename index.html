1. What does CSS Stand For

    Cascading Style Sheets

2. Which of the following is correct?

    jQuery is a JavaScript Library

3. jQuery Uses CSS selectors to select elements

    True

4. Look at the following selector: $("div") What does it select?

    All div elements

5. Is jQuery a library for client side or server side?

    Client side

6. What is the correct jQuery code to set the background color of all p elements to red?

    $("p").css("background-color","red");

7. Which jQuery method is used to perform an asynchronous HTTP request?

    jQuery.ajax()

8. Which jQuery function is used to prevent code from running, before the document is finished loading?

    $(document).ready()

9. Look at the following selector: $("div p"). What does it select?

    All p elements inside a div element

10. The jQuery animate() method can be used to animate ANY CSS property?

    Only properties containing numeric values
